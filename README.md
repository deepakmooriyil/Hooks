# 100 Viral Hooks — Swipe File

A searchable swipe file of 100 viral content hooks with the psychology behind each one.

## Features

- 100 hooks across 10 psychological categories
- Search and filter by category
- One-click copy any hook
- "Why it works" toggle for each hook
- Zero dependencies — pure HTML/CSS/JS

## Deploy to Vercel

### Option 1 — Vercel Dashboard (easiest)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Click **Deploy** — done ✅

### Option 2 — Vercel CLI
```bash
npm i -g vercel
vercel
```
